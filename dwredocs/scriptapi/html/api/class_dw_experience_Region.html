Class Region

dw.experience.Region
=========================== EXPERIENCE BUILDER BETA API =========================== This class represents a region which serves as container of components. Using the PageMgr.renderRegion(Region) or PageMgr.renderRegion(Region, RegionRenderSettings) a region can be rendered.
