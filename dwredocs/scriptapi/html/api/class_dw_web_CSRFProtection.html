dw.web.CSRFProtection
Used to generate and validate CSRF tokens. CSRFProtection allows applications to protect themselves against CSRF attacks, using synchronizer tokens, a best practice. Once created, these tokens are tied to a userâ€™s session and valid for 60 minutes.

Usage:
Adding CSRF token to forms: 

 //CSRF token generation
 <form ... action="">
   <input name="foo" value="bar">
   <input name="${dw.web.CSRFProtection.getTokenName()}"
             value="${dw.web.CSRFProtection.generateToken()">
 </form>
Then, in scripts call: 
 dw.web.CSRFProtection.validateRequest();
