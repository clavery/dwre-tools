Class Object
Object
The Object object is the foundation of all native JavaScript objects. Also, the Object object can be used to generate items in your scripts with behaviors that are defined by custom properties and/or methods. You generally start by creating a blank objcet with the constructor function and then assign values to new properties of that object.
