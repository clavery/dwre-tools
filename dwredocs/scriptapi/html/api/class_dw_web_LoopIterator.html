Class LoopIterator
Object
dw.util.Iterator
dw.web.LoopIterator
Iterator used in <ISLOOP> implementation. It defines properties used to determine loop status. LoopIterator object is assigned to variable declared in "status" attribute of the <ISLOOP> tag.
