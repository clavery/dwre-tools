dw.experience.Page
 ===========================
 EXPERIENCE BUILDER BETA API
 ===========================
This class represents an experience builder managed page. A page comprises of multiple regions that hold components, which themselves again can have regions holding components, i.e. spanning a hierarchical tree of components. Using the PageMgr.renderPage(Page) a page can be rendered. As such page implements a view function for creating render output the view function of the page itself will also want to access its various properties like the SEO title etc. Furthermore the portion of code that triggers rendering of the page (likely a controller like Page-Show) is also able to check the current visibility of the page (isVisible()) in order to react accordingly.
