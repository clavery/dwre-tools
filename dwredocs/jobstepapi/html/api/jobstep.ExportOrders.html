Overview
Job Step ExportOrders
Exports order data. The Search Service is used to find the orders.
Execution Context:
Sites
Parallel Execution:
Not Supported
Always execute on restart:
Supported
Input Parameters
Canceled  :  Boolean (Optional)
The 'canceled' order status.
Default Value: true
Completed  :  Boolean (Optional)
The 'completed' order status.
Default Value: true
Confirmation Status  :  String (Required)
The 'confirmation' status.
Allowed Values: All, Not Confirmed, Confirmed
Default Value: All
Created  :  Boolean (Optional)
The 'created' order status.
Default Value: true
Export Status  :  String (Required)
The 'export' status.
Allowed Values: All, Not Exported, Exported, Ready for Export, Export Failed
Default Value: All
ExportFile  :  String (Optional)
Export file pathname relative to 'IMPEX/src' folder. Either ExportFile or FileNamePrefix must be specified.
Failed  :  Boolean (Optional)
The 'failed' order status.
Default Value: true
FileNamePrefix  :  String (Optional)
The file name prefix of the export file. The job step will append site information where relevant and a timestamp. Sorting the export files alphanumerically will yield the chronological order. The parameter may include a folder component. Path is relative to 'IMPEX/src' folder. Either ExportFile or FileNamePrefix must be specified.
New  :  Boolean (Optional)
The 'new' order status.
Default Value: true
Open  :  Boolean (Optional)
The 'open' order status.
Default Value: true
OverwriteExportFile  :  Boolean (Optional)
Overwrite existing file. If option is not selected and the file already exists, the job step will use the ERROR exit status.
Default Value: true
Payment Status  :  String (Required)
The 'payment' status.
Allowed Values: All, Not Paid, Partially Paid, Paid
Default Value: All
Replaced  :  Boolean (Optional)
The 'replaced' order status.
Default Value: true
Shipment Status  :  String (Required)
The 'shipment' status.
Allowed Values: All, Not Shipped, Partially Shipped, Shipped
Default Value: All
Use Order Export Delay  :  Boolean (Optional)
See 'Order Export Delay' at 'Site Preferences' -> 'Order Preferences' -> 'Order Export Settings'
Default Value: false
Exit Status
OK
Step exit status used when the export finished successfully.
ERROR
Step exit status used when the export failed.
